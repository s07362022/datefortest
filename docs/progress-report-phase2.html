🔌 全部 API 端點（15 個）

Auth — 認證

POST/auth/register註冊帳號，回傳 JWT token
POST/auth/login登入，JSON body: {email, password}
GET/auth/me查看自己的帳號資料

Profiles — 個人檔案

GET/profiles/me查看自己的完整檔案（含完成度）
PUT/profiles/me更新檔案（顯示名稱、生日、性別、城市、簡介、標籤）
GET/profiles/{user_id}查看指定用戶的公開檔案

Preferences — 配對偏好

GET/preferences/me查看自己的偏好設定
PUT/preferences/me更新偏好（性別、年齡範圍、最大距離）

Photos — 照片 Phase 2 新增

POST/photos上傳照片（multipart/form-data，JPEG/PNG/WebP，≤5MB，≥400×400px）
GET/photos/me查看自己所有照片（依 sort_order 排序）
DELETE/photos/{photo_id}刪除照片（自動重排序，若刪主照片則自動換下一張）
PUT/photos/{photo_id}/primary設定主照片（封面）
GET/photos/file/{filename}下載/瀏覽照片（需認證）

Tags — 興趣標籤 Phase 2 新增

GET/tags列出全部 30 個預設興趣標籤

Discover — 推薦探索 Phase 2 新增

GET/discover?limit=20&cursor=推薦對象列表（需完整個人檔案）

🎯 推薦算法

總分 = Profile完整度 × 0.4 + 活躍度分數(0-20) + 距離分數(0-20) + 共同興趣分數(0-20)
評分項目滿分規則
Profile 完整度100 × 40%display_name / birthday / gender / interested_in / city / bio≥20字 / 有照片 / ≥3標籤
活躍度201h內=20, 24h內=15, 72h內=10, 1週內=5, 更久=0
距離20越近越高分，超過 max_distance_km 自動排除
共同興趣20每個共同標籤 +5 分（上限 20）

🚫 自動排除：自己 / 已封鎖用戶（Phase 6 實裝）/ 已 Like 或 Pass 過的用戶（Phase 4 實裝）

🧪 如何測試（Swagger UI 操作指南）

開啟瀏覽器前往：http://localhost:8000/docs

【測試情境 A】用 Seed 帳號登入並探索

登入 Seed 帳號 — 點 POST /auth/login → 輸入 {"email":"alice@test.com","password":"Seed1234!"} → 複製 access_token
授權 — 點右上角 Authorize 🔒，貼上 token，格式：Bearer eyJ...
查看個人檔案 — GET /profiles/me → 可看到 Alice 的完整資料與標籤
探索推薦 — GET /discover?limit=5 → 回傳其他符合條件的用戶（依評分排序）
查看標籤清單 — GET /tags → 列出全部 30 個興趣標籤（可用 id 更新個人標籤）

【測試情境 B】新帳號上傳照片

註冊新帳號 — POST /auth/register → {"email":"newuser@test.com","password":"Test1234!"}
取得 token 後授權（同上 Authorize）
上傳照片 — POST /photos → 點 Try it out → Choose File → 選一張 JPG（≥400px）→ Execute
查看照片 — GET /photos/me → 可看到剛上傳的照片資訊
更新個人檔案 — PUT /profiles/me → 設定完整資料（顯示名稱、生日、性別、簡介、tag_ids）
再測試 Discover — 完整個人檔案後才能呼叫 GET /discover，否則會回傳 403 PROFILE_INCOMPLETE

【測試情境 C】用 PowerShell 快速驗證

登入 Seed 帳號取得 token：
$r = Invoke-RestMethod -Method Post -Uri "http://localhost:8000/auth/login" -ContentType "application/json" -Body '{"email":"bob@test.com","password":"Seed1234!"}'
呼叫 Discover：
Invoke-RestMethod -Uri "http://localhost:8000/discover?limit=3" -Headers @{Authorization="Bearer $($r.access_token)"} | ConvertTo-Json -Depth 5
查看 Tags：
Invoke-RestMethod -Uri "http://localhost:8000/tags" -Headers @{Authorization="Bearer $($r.access_token)"}
